# FizzBuzz

Rozwiązanie zadania FizzBuzz w C#.

Program wypisuje do konsoli liczby od 0 do zadanej liczby:

 - Kiedy liczba jest podzielna przez 3: **Fizz**
 - Kiedy liczba jest podzielna przez 5: **Buzz** 
 - Kiedy liczba jest podzielna przez 3 i 5: **FizzBuzz**
- Gdy liczba nie jest podzielna przez 3 lub 5, program wypiszę **liczbę**.
