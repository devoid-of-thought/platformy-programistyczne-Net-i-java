# Platformy Programistyczne .NET & Java

Repozytorium z rozwiązaniami zadań laboratoryjnych do kursu "Platformy Programistyczne .NET i Java" na Politechnice Wrocławskiej.

Metoda utworzenia nowej aplikacji konsolowej bez użycia top-level-statements w Visual Studio Code:

1. Stworzenie nowego folderu.

2. Przejście do nowego folderu i wykonanie komendy
```
dotnet new console --use-program-main
```
